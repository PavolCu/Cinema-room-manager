The sequence

You are given a sequence of natural numbers. Find the maximum element divisible by 4. The input is passed to the program as follows:

  1. Number of elements in the sequence

  2. Elements of the sequence

In the sequence, there is always an element divisible by 4. The program should print a single number: the maximum element of the sequence divisible by 4.

Try to solve this problem by using a while-loop.

Note: The first number passed to the program (number of elements) is not included in the sequence.


Sample Input:
8
16
49
91
77
36
12
12
36

Sample Output:
36



Memory limit: 256 MB
Time limit: 5 seconds
