Grading System: Functional Decomposition

In the context of Java programming and functional decomposition, given the following code snippet, which of the following statements are true? 

public class GradingSystem {

    public static void main(String[] args) {
        int score = 85;
        computeGrade(score);
    }

    public static char computeGrade(int score) {
        char grade;
        switch (score / 10) {
            case 10: case 9:
                grade = 'A';
                break;
            case 8:
                grade = 'B';
                break;
            default:
                grade = 'C';
        }
        return grade;
    }
}
